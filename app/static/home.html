How it works

Three simple steps

Get from zero to instant address lookups in minutes.

1

Create a project

Organise your work into projects — one per city, service region, or client. Create as many as you need.

2

Upload your polygons

Drop in a GeoJSON or KML file that defines your coverage zones. Multiple polygons per project are supported.

3

Check any address

Features

Everything you need

Built for teams that need reliable, fast coverage lookups.

Multiple projects

Create separate projects for different service areas, clients, or campaigns. Each has its own polygons and check history.
